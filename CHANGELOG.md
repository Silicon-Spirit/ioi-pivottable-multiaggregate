# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-14

### Added
- Initial release of Vue3 PivotTable for Frappe Framework
- Complete Vue 3 project structure with proper build configuration
- Main components: Pivottable, PivottableUi, TableRenderer
- Support for multiple renderers (Table, Heatmaps, Charts, Export)
- Rich set of aggregators (Count, Sum, Average, etc.)
- Drag-and-drop interface for configuring pivot tables
- Frappe Framework integration with dedicated entry point
- Vite build configuration for ES, CJS, and UMD formats
- Comprehensive documentation and examples

### Structure
- Organized source code in `src/` directory
- Components in `src/components/`
- Utilities in `src/utils/`
- Styles in `src/styles/`
- Entry points in `src/entry/`

### Build
- Vite-based build system
- Separate builds for general use and Frappe Framework
- CSS bundling support
- Source maps for debugging
